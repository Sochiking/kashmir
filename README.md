# Kashmir Global Ecommerce Starter

A responsive Next.js storefront for a phone and accessories company.

## Features
- Light-blue Kashmir Global branding
- Responsive homepage and mobile navigation
- Product categories and search
- Product detail pages
- Cart drawer with quantity controls
- WhatsApp checkout
- Locally stored placeholder product artwork
- SEO metadata and App Router structure

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Customisation
- Update products in `data/products.ts`
- Replace SVG files in `public/products`
- Change WhatsApp number in `components/Storefront.tsx` and product page
- Change contact and location information in the footer

## Production upgrades
Connect a real database, admin dashboard, authentication, payment gateway, delivery API and cloud image storage before accepting automated online payments.
